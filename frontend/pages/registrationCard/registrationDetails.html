Operating Systems (CMPS 405)
Course Details:

Title: Operating Systems

Associated Term: Spring 2025

Crediet Hours: 3

College: Engineering

Major: Computer Science

This course provides a comprehensive introduction to operating systems, covering fundamental concepts such as process management, memory management, file systems, and concurrency. Students will explore how operating systems allocate resources, schedule processes, manage input/output operations, and ensure security and protection. The course also delves into topics such as virtualization, multithreading, deadlock handling, and system performance optimization.
